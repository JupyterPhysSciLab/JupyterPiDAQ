**JupyterPiDAQ**

This software allows collection and plotting of analog data
collected using an ADS1115 A-to-D piHAT on a Raspberry Pi.
So far this has been tested on a RPi 3B+.

Author: Jonathan Gutow <jgutow@new.rr.com>
